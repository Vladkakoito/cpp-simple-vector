Простой Вектор

Упрощенная версия std::vector, тренирова работы с умным указателем, с динамической памятью

Поддерживаемые методы PushBack, PopBack, Insert, Resize, Reserve, operator[], At, конструкторы по аналогии с std::vector, а так же работа с итераторами.

Методы, которые это подразумевают, поддерживают rvalue-ссылки
